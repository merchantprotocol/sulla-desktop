# UI Libraries Used

This project uses the following open-source UI libraries for enhanced functionality:

- **Monaco Editor** ([https://microsoft.github.io/monaco-editor/](https://microsoft.github.io/monaco-editor/)): Powers the code editing experience in the filesystem viewer and diff editor.
- **Vue Flow** ([https://vueflow.dev/](https://vueflow.dev/)): Powers the node-graph workflow editor.
- **Mantine UI** ([https://mantine.dev/](https://mantine.dev/)): Component library for UI elements.
- **xterm.js** ([https://xtermjs.org/](https://xtermjs.org/)): Terminal emulator for the integrated terminal.
- **Preline** ([https://preline.co/docs/context-menu.html](https://preline.co/docs/context-menu.html)): Provides context menu components.
